Timeline of the Internet

Year Event
1969 ARPANET, the first network to use the Internet protocol suite, was created.
1983 Domain Name System (DNS) introduced.
1989 Tim Berners-Lee proposes the World Wide Web.
1991 First website launched by CERN.
1993 Mosaic, the first popular web browser, was released.
1998 Google was founded.
2004 Facebook was launched.
2005 YouTube was created.
2007 Apple launched the iPhone, increasing mobile internet usage.
2010 Instagram launched, marking rise of visual social platforms.
2020 Global internet usage surpasses 4.5 billion users.
